# runPas-FC
tiny bash script and toolchain to run Pascal-FC code

# Usage
1) clone the repository (pccomp is the compiler, pint the interpreter, as binaries)
2) run the script with your .pas file as an argument (./runPas.sh yourCode.pas [-k]
OPTIONAL: Use -k to keep the files generated by the compiler and interpreter
3) type :q and press ENTER to exit the interpreter. It will clean the temp files.
